# ConTroll Version 1.5 Release Notes

## Version 1.5: Marketing/More Custom Text/Terminal Tracking, :PDF Badge Printing, Two Line Badge Names,  Multiple Sales Taxes
### Release Date: December 1, 2025

# Major Configuration Changes in 1.5:

* **ConTroll now requires PHP 8.3 due to the dependencies of it's add-on's.**

* New Database Patches   
  * 51: Art Show: Cleanup of Exhibitors tables, Addition of more configurable text for registration related emails
  * 52: More Custom Text: Cleans up dedup of policies and interests, Cleans up merging people, Addition of more configurable text for registration related 
    emails, Terminal payment tracking changes
  * 53: More Terminal: PDF Badge Printing, Convention Role tables, more on merging people, minor size change in artSales
  * 54: More Custom Text: Adds more exhibitor custom text fields, Two Line Badge Labels
  * 55: Multiple Taxes: adds tax1-5 for taxable items, and some minor membership config table data changes.
      
* New/Changed/Deleted Config File Entries:  
  * Enforced Splitting of the config file into three sections, see System-Documents-and-Coding-Standards/Configuration-File-Format-and-Usage for details.
  * 1.5 Loads the three files in the order reg_conf.ini, reg_admin.ini and finally reg_secret.ini.  reg_admin.ini will overwrite any fields aleady 
  loaded by reg_conf.ini. reg_secret will overwrite any section already loaded.
  * Addition of better documentation in the sample config files.
  * Addition of all the hinting fields for reg_conf.ini.sample to drive web based editing of reg_conf.ini
  * reg_admin.ini:
    * depreciated badgeps (no longer used in PDF printing)
    * added badgeLogo: logo image (png or jpg) for the badge
  * reg_secret.ini:
    * added comment default lines for the new multi-location by application for Square:

            ; location_portal=""
            ; location_artist=""
            ; location_vendor=""
            ; location_exhibithall=""
            ; location_fantable=""
            ; location_controllreg=""
            ; location_controllexhibits=""
            ; location_atcon_regpos=""
            ; location_atcon_artpos=""
  
  * reg_conf.ini  
    * posRegistrationSort: how to sort the membership buttons in Mail in Reg and atcon (POS based registration)
    
* New Scripts: None
